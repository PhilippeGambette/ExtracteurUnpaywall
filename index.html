Extracteur Unpaywall

Cet outil vise à évaluer la proportion de publications en libre accès parmi une liste de DOI, en utilisant les API d'Unpaywall et de HAL.

Codé par Philippe Gambette en Javascript/jQuery, il est dérivé de l'outil conçu par Frédérique Bordignon et Romain Boistel, présenté sur le carnet Hypothèses Carnet'IST. Son code est disponible sur GitHub sous licence GPLv3.
